## Goal

Remove low-risk unused leftovers identified by architecture review 0023 without
changing production behavior.

## Context

Architecture review 0023 found a first cleanup group that is small enough to
verify independently: unused panel state, unused settings fields, and a
template-like smoke test filename.

## Scope

- Remove unused `FilePanelState.FilterMask`.
- Remove unused `AppSettings.Ui.Theme`.
- Remove unused `AppSettings.Shell.Kind`.
- Rename the generic `UnitTest1.cs` smoke test file to a purpose-revealing name.

## Non-goals

- Do not change runtime behavior.
- Do not add settings compatibility code.
- Do not extract shared dialog/list helpers.
- Do not change the panel provider or search-result source model.

## Acceptance criteria

- The codebase has no references to `FilterMask`, `Ui.Theme`, or `Shell.Kind`.
- The smoke tests are in a file with a purpose-revealing name.
- Repository checks pass.

## Verification

Commands:

```bash
dotnet build
dotnet test
```
