# 0036 - Application icon

Related:
- none

## Goal

Give CSharpFar its own Windows application icon with a Far-like file manager
identity without copying Far Manager branding or artwork.

## Context

The executable currently has no project icon configured. The application should
look identifiable in Windows shells, task switchers, and published artifacts.

## Scope

- Add an original icon asset for CSharpFar.
- Use a compact, high-contrast design that suggests a two-panel console file
  manager.
- Include icon sizes appropriate for a Windows executable.
- Configure the application project to embed the icon into the produced
  executable.

## Non-goals

- Do not copy the Far Manager icon, logo, exact layout, or trademarked visual
  details.
- Do not add installer branding, file associations, shortcuts, or packaging
  changes.
- Do not change runtime UI rendering or application behavior.
- Do not add compatibility paths around icon generation.

## Acceptance criteria

- The repository contains the icon source and `.ico` asset.
- `CSharpFar.App` embeds the icon into `CSharpFar.exe`.
- The compiled executable exposes an icon resource.

## Verification

Manual checks:

1. Inspect the generated executable icon in the output directory.
