# 011. Диалог создания папки

Type: task
Status: done

Related:

## Goal

Сделать диалог `F7` создания папки визуально ближе к Far-style окнам файловых операций.

## Context

`F7` сейчас использует простой `InputDialog` с маленьким окном и текстом `Folder name:`. Окна copy/move уже используют общую модальную инфраструктуру с внешней подложкой, внутренней рамкой и общим button bar.

Нужно оставить только сценарий создания папки: без link type, target и поддержки нескольких имен.

## Done when

- `F7` показывает заголовок `Make folder`.
- Подпись поля ввода: `Create the folder:`.
- Диалог использует `ModalDialogRenderer` и `DialogButtonBar`.
- Внутренняя рамка двойная.
- Кнопки: `{ OK }` и `[ Cancel ]`.
- Существующая валидация и создание вложенных каталогов остаются рабочими.
